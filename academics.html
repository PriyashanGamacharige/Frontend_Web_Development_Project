Academic Programs

Undergraduate Programs

  • BSc in Information Technology (4 years)

Our flagship program provides comprehensive training in software development, systems analysis, and IT project management. Students gain hands-on experience with modern technologies through our industry partnerships.

Key Features: Specializations in Cloud Computing or Mobile Development, Industry certifications included, 6-month internship program

Career Paths: Software Developer, IT Consultant, Systems Analyst, Project Manager

  • BSc in Computer Science (4 years)

A rigorous program focusing on algorithms, data structures, and computational theory. Prepares students for advanced research or high-performance computing careers.

Key Features: Advanced mathematics foundation, AI and machine learning electives, Research opportunities from year 2

Career Paths: Software Engineer, Data Scientist, Research Scientist, Algorithm Specialist

  • BSc in Data Science (4 years)

Combines statistics, computer science, and business intelligence to create experts in extracting insights from big data.

Key Features: Access to our Big Data Lab, Real-world datasets for projects, Industry-standard tools (Hadoop, Spark, TensorFlow)

Career Paths: Data Analyst, Business Intelligence Specialist, Machine Learning Engineer

  • BSc in Cyber Security (4 years)

Develops expertise in protecting digital assets and infrastructure from cyber threats through ethical hacking, digital forensics, and security architecture.

Key Features: Certified Ethical Hacker (CEH) preparation, Cyber range for realistic training, Partnerships with security firms

Career Paths: Security Analyst, Penetration Tester, Chief Information Security Officer

Postgraduate Programs

  • MSc in Information Technology (2 years)

Advanced study of enterprise IT systems, emerging technologies, and strategic IT management for professionals seeking leadership roles.

Key Features: Evening/weekend classes for working professionals, Capstone project with industry mentor, Focus on digital transformation

Career Paths: IT Director, Chief Technology Officer, Solutions Architect

  • MSc in Artificial Intelligence (2 years)

Cutting-edge program covering machine learning, neural networks, and cognitive computing with applications across industries.

Key Features: Access to GPU computing cluster, Research partnerships with AI labs, Specializations in Computer Vision or NLP

Career Paths: AI Engineer, Machine Learning Specialist, Robotics Programmer

  • PhD in Computer Science (3-5 years)

Research-intensive program for those pursuing academic careers or advanced industrial research positions.

Key Features: Individualized research plans, International collaboration opportunities, Teaching assistantships available

Career Paths: University Professor, Research Scientist, Chief Innovation Officer

Diploma Programs

  • Diploma in Software Engineering (2 years)

Practical training in full-stack development, software testing, and agile methodologies for immediate industry entry.

Key Features: Project-based learning, Portfolio development, Industry-aligned curriculum updated annually

Career Paths: Junior Developer, QA Engineer, DevOps Technician

  • Diploma in Network Engineering (2 years)

Comprehensive training in network design, implementation, and security for enterprise environments.

Key Features: Cisco Networking Academy partner, CCNA preparation, Hands-on lab with enterprise equipment

Career Paths: Network Administrator, Systems Engineer, Cloud Network Specialist

  • Diploma in Web Development (1 year)

Intensive program covering modern web technologies and frameworks for front-end and back-end development.

Key Features: MERN stack focus, Responsive design principles, 3-month internship included

Career Paths: Web Developer, UI/UX Developer, Front-end Engineer

Certificate Programs

  • Certificate in Programming (6 months)

Foundational programming skills in Python and JavaScript with introduction to algorithms and problem-solving.

Key Features: Beginner-friendly, Project portfolio creation, Pathway to diploma programs

Career Paths: Junior Programmer, Technical Support Specialist

  • Certificate in Database Management (6 months)

Practical skills in database design, SQL, and administration using MySQL, Oracle, and MongoDB.

Key Features: Real-world database projects, Performance optimization techniques, Data modeling fundamentals

Career Paths: Database Administrator, Data Analyst, BI Developer

  • Certificate in UI/UX Design (6 months)

Comprehensive training in user-centered design principles, prototyping tools, and usability testing.

Key Features: Adobe XD and Figma training, Design thinking methodology, Portfolio development

Career Paths: UI Designer, UX Researcher, Interaction Designer
